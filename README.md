## Questions Covered

### Question 1 – Linux Environment Verification
- Verified username and groups
- Checked current shell
- Displayed current working directory
- Listed files and directories
- Verified network connectivity

### Question 2 – Secure Project Workspace Setup
- Created project workspace
- Configured file permissions
- Verified ownership
- Observed umask settings
- Documented security considerations

### Question 3 – File System and Link Analysis
- Created regular file
- Created hard link
- Created symbolic link
- Compared inode information
- Studied link behavior after deleting the original file

### Question 4 – File Access and I/O Investigation
- Investigated open files
- Observed file descriptors
- Practiced output and error redirection
- Checked resource limits
- Documented Linux file I/O behavior

### Question 5 – Storage Health Assessment
- Listed storage devices
- Examined mounted file systems
- Checked disk usage
- Verified inode utilization
- Documented storage health observations and recommendations
